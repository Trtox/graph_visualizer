EdgesUI

class EdgesUI(val onEdgesChange: (Set<Edge>) -> Unit, vertices: VerticesUI)

User interface component for managing and displaying a list of graph edges.

This class provides a text area where users can input graph edges in the format: "A -> B", where A and B are vertex labels. The edges are parsed and managed by the EdgesManager.

The class listens for changes to the text area and updates the graph's edges accordingly. It also reacts to changes in the status of vertices by enabling or disabling related edges through the EdgesManager.

The EdgesUI component interacts with the VerticesUI component to ensure consistency between the vertices and edges. It also notifies the parent class of changes in the edges via the provided onEdgesChange callback.
