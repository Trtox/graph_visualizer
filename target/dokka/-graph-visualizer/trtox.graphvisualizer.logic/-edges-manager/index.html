EdgesManager

class EdgesManager(val onEdgesChange: (Set<Edge>) -> Unit, vertices: VerticesUI)

Manages the edges of the graph and updates them based on changes in the edge text or the status of vertices. It interacts with the VerticesUI class to ensure all vertices are valid when creating edges and updates the graph accordingly. It also triggers the onEdgesChange callback whenever the set of enabled edges changes.

The class is responsible for adding, removing, and updating edges. It ensures that any vertices involved in an edge are valid and updated. The edges are stored in an internal Edges object.
